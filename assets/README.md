# 🖼️ Assets

> Static assets used across the repository — diagrams, logos, banners, and visual materials.

---

## What Lives Here

The `assets/` folder stores all visual and static files referenced across README files and documentation.

```
assets/
├── diagrams/       ← Architecture and flow diagrams
├── screenshots/    ← System screenshots for READMEs
├── logos/          ← Logos and brand marks
└── banners/        ← Repository banner images
```

---

## Asset Types

### 📐 Diagrams
Architecture diagrams, data flow charts, and system maps.

- Exported as `.png` or `.svg`
- Referenced in architecture docs and system READMEs
- Created with draw.io, Mermaid, or Lucidchart

### 📸 Screenshots
Captured screenshots of running systems.

- n8n workflow views
- Google Sheets output samples
- Telegram bot interactions
- API response examples

### 🎨 Logos & Banners
Visual branding for the repository.

- Repository banner for root README
- Tool and platform logos (n8n, OpenAI, Supabase, etc.)
- Social preview image

---

## Naming Convention

```
[folder]/[system-or-topic]-[descriptor].[ext]

Examples:
diagrams/google-lead-generator-flow.png
screenshots/n8n-workflow-overview.png
banners/repo-header.png
```
