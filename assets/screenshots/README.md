# 📸 Screenshots

> System screenshots for README files, documentation, and demo materials.

---

## What Lives Here

This folder contains screenshots of live running systems — captured at key moments to document what each system looks like in action.

Screenshots are used in:
- System `README.md` files to show output
- `demos/` walkthroughs
- Portfolio and social media content

---

## Screenshot Categories

### 🔄 n8n Workflow Views
Screenshots of n8n workflow canvases showing the full node structure.

```
n8n-[system-name]-overview.png
n8n-[system-name]-detail-[section].png
```

### 📊 Output Samples
Screenshots of actual system output — Google Sheets, Telegram messages, API responses.

```
output-[system-name]-sheets.png
output-[system-name]-telegram.png
output-[system-name]-api-response.png
```

### ⚙️ Configuration Views
Screenshots of API settings, credential configs, and node settings.

```
config-[system-name]-[component].png
```
