## File Format Guidelines

| Format | Use Case |
|--------|---------|
| `.svg` | Preferred — scales without quality loss |
| `.png` | For markdown embeds (GitHub renders PNG best) |
| `.ico` | Favicon if needed |

---

## Sizing Standards

```
Icon (square):   128×128px minimum, 512×512px preferred
Logo (wide):     Any width, 60-80px height
Badge:           20px height (shield.io compatible)
```
