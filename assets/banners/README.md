# 🖼️ Banners

> Full-width banner images for this repository and its systems.

---

## What Lives Here

This folder contains banner and header images used in README files and on social media platforms.

Banners are the first visual impression of the repository — they should be sharp, professional, and on-brand.

---

## Banner Types

### 🚀 Repository Banner
The main header image shown at the top of the root `README.md`.

```
repo-header.png         ← Main repository banner (dark theme)
repo-header-light.png   ← Light theme variant
repo-header.svg         ← SVG version for scalability
```

### ⚙️ System Banners
Per-system banners used at the top of each system's `README.md`.

```
[system-name]-banner.png

Examples:
google-lead-generator-banner.png
semantic-db-telegram-banner.png
youtube-trending-search-banner.png
```

### 📱 Social Preview
GitHub repository social preview image (shown when sharing on Twitter, LinkedIn, etc.)

```
social-preview.png   ← 1280×640px (GitHub recommended size)
```

---

## Size Specifications

| Type | Dimensions | Format |
|------|-----------|--------|
| Repository header | 1200×300px | PNG / SVG |
| System banner | 1200×200px | PNG |
| Social preview | 1280×640px | PNG |
| GitHub profile README banner | 1200×400px | PNG |

---

## Design Guidelines

- **Background:** Dark (`#0d1117`) or gradient preferred
- **Font:** Bold, clean, tech-forward
- **Colors:** Consistent with repo theme (deep blues, purples, or monochrome)
- **Text:** Repository name + short tagline
