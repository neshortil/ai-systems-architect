# 🎬 Demos

> Live previews, screen recordings, and visual walkthroughs of every system in this repo.

---

## What Lives Here

Each demo folder contains:

- **Screen recordings** — MP4 or GIF of the system running live
- **Screenshots** — key moments from the workflow execution
- **Sample outputs** — example data produced by the system
- **Setup walkthrough** — step-by-step video or written guide

---

## Available Demos

| System | Type | What You'll See |
|--------|------|-----------------|
| Google Lead Generator | Video + Screenshots | Workflow run → leads in Google Sheets |
| Instagram Lead Generator | Screenshots | Profile scrape → enriched data output |
| YouTube Trending Search | Screenshots | Search → ranked results in Sheets |
| Semantic DB + Telegram | Video | Query → AI response via Telegram bot |

---

## Demo Format

All demos follow this structure:

```
demos/
└── [system-name]/
    ├── demo.mp4           ← Full walkthrough video
    ├── screenshots/       ← Key moments captured
    └── sample-output/     ← Example data files
```
