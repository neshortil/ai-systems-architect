# 🚀 Services

## 🚀 Starter — Single Automation System

**Best for:** Businesses with one clear manual process to automate.

**What you get:**
- 1 end-to-end automation workflow
- API integrations (2–3 services)
- Basic AI integration (classification, generation, or extraction)
- Telegram or webhook-based control interface
- Documentation and handoff

**Examples:**
- Automated lead collection from Google Maps
- Email enrichment and personalization pipeline
- Data extraction and formatting from any source
- Automated report generation and delivery

**Timeline:** 1–2 weeks
